Kern Programming Language

Python-like productivity. C++-level system access.

Readable scripts compiled to bytecode, executed by a fast VM. Built for developers who want familiar syntax with explicit control over process, filesystem, networking, and memory.

Download Portable

Features

Everything you need for systems programming and scripting

Python-Like Syntax

Familiar statements, functions, classes, and modules. No boilerplate—just write code that runs.

Fast Bytecode VM

Compiles to bytecode and executes on a high-performance VM. Get the speed you need without sacrificing readability.

C++-Level System Access

Direct access to filesystem, processes, networking, and memory. Built for systems automation and real-world scripting.

Inspectable Pipeline

See what's happening under the hood with --ast, --bytecode, and --check. No black-box behavior.

2D & 3D Graphics

Optional Raylib-backed graphics with g2d, g3d, and a built-in 3D engine with ECS. Build games and visualizations.

Package Manager (Kargo)

GitHub-first package management with kargo. Install, publish, and manage dependencies with semver support. (currently broken)

Write Kern Code

From scripts to systems to games

hello.kn 
// No main() required - statements run at top level
print("Hello, Kern!")

// Variables and expressions
let name = "developer"
print("Welcome,", name)
// Read and write files with built-in functions
let content = readFile("input.txt")
print("File contents:", content)

// Write to file
writeFile("output.txt", "Hello from Kern!")
print("File written successfully")
// Fetch data from the web
let http = import("std.v1.http")

let response = http.get("https://api.github.com/repos/entrenchedosx/kern")
print("Stars:", response["stargazers_count"])
// 3D game with Kern
def main() {
    let g3 = import("g3d")
    
    g3.createWindow(960, 540, "Kern 3D Demo")
    g3.setCamera(6.0, 4.0, 6.0, 0.0, 0.5, 0.0)
    
    let cube = g3.createObject("cube", 1.8)
    g3.setObjectColor(cube, 255, 80, 80)
    
    let angle = 0.0
    
    let update = lambda () => {
        angle = angle + 0.02
        g3.setRotation(cube, 0.0, angle, 0.0)
    }

    let draw = lambda () => {
        // Rendering handled automatically
    }

    g3.run(update, draw)
}

Quick Start

Get up and running in 3 steps

1

Download

Extract the portable archive. No installer needed.

kern-v2.0.1-windows-x64.zip
2

Write

Create a .kn file. No main() required.

print("Hello, Kern!")
3

Run

Execute with the Kern VM.

./kern.exe myscript.kn

Language Reference

Core syntax at a glance

Statements

  • let / var / const
  • def functions
  • class types
  • return, if / else
  • while, for, match
  • try / catch / defer

Values

  • Numbers, strings, booleans
  • nil (null)
  • Arrays [a, b]
  • Maps { "key": value }
  • Lambda expressions
  • Function references

Operators

  • Math: + - * / %
  • Compare: == != < >
  • Logic: && || !
  • Optional: ?.
  • Null coalesce: ??

Modules

  • import("math")
  • import("lib/kern/foo")
  • from "mod" import fn
  • Built-in: g2d, g3d
  • Stdlib: std.v1.*
